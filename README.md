ConstructionXpert Services Solution
Description du Projet
ConstructionXpert Services Solution est une application web permettant à une équipe de construction de créer et de gérer des projets de construction, d'assigner des tâches aux membres de l'équipe, de gérer les ressources nécessaires et de suivre les détails des projets et des tâches. Cette application vise à fournir à l'équipe de construction un outil efficace pour planifier et organiser les projets de construction tout en garantissant la sécurité des données et l'intégrité du système.

Fonctionnalités Principales
Gestion des Projets : Créer, afficher, mettre à jour et supprimer des projets avec leurs détails.
Gestion des Tâches : Créer, afficher, mettre à jour et supprimer des tâches associées à un projet avec leurs détails.
Gestion des Ressources : Ajouter, afficher, mettre à jour et supprimer des ressources avec leurs détails.
Technologies Utilisées
Frontend : HTML5, CSS (Bootstrap/Tailwind), JavaScript
Backend : Java (Java EE, Servlets, JSP)
Base de Données : MySQL ou PostgreSQL
Outils : Eclipse/IntelliJ IDEA, Maven, Git
Autres : UML pour la conception, Figma pour les maquettes, Apache Tomcat comme serveur d'application
Installation et Utilisation
Clonez ce dépôt sur votre machine locale.
Importez le projet dans votre IDE (Eclipse/IntelliJ IDEA).
Configurez la base de données en utilisant MySQL ou PostgreSQL.
Compilez et déployez l'application sur un serveur Apache Tomcat.
Accédez à l'application via votre navigateur web préféré.
